## Course Structure
Each day's lesson is in a separate folder, containing:
- `README.md` (Theory + Examples)
- `.py` scripts (Runnable Examples)
- `exercises.py` (Practice Tasks)

## How to Use This Repository?

1️⃣ Clone the repo:

2️⃣ Navigate to a day's folder:
